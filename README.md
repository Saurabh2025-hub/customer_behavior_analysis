# customer_behavior_analysis
Data analytics project showcasing customer behavior analysis using python, sql and power BI.
This project shows a complete data analysis process using Python, SQL, and Power BI. It includes cleaning and exploring data in Python, analyzing it using SQL, and creating an interactive dashboard in Power BI. The goal is to understand customer behavior like spending patterns, product performance, discounts, and customer types. Overall, it shows how raw data can be turned into useful insights to support business decisions.
